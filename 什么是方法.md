# 何为方法
> System.out.println
#### System是一个系统的类，out是一个输出对象，println就是一个方法
#### 方法是一个语句的集合，他们在一起执行一个功能
1. Java方法名是驼峰命名，eg。valueType
2. Java方法需要修饰符，返回值类型，方法名（参数类型 参数名）{
}
3. 方法体包含具体的语句，定义该方法的功能
# 方法的重载
> 方法的重载规则
1. 主要在于方法类型的不同，有相同的函数名称，但是形参不相同
2. 方法名必须相同
3. 参数列表必然不同
4. 方法的返回类型可以相同也可以不同
5. 仅仅返回类型不同不足以成为方法的重载
